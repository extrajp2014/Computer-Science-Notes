## Most Common Commands

| Command                   | Description                           |
| ------------------------- | ------------------------------------- |
| `code .`                  | Open current folder                   |
| `code filename.py`        | Open a file                           |
| `code -g file.py:42`      | Open file at line 42                  |
| `code -n .`               | Open in a new window                  |
| `code -r .`               | Reuse the current window              |
| `code --wait file.txt`    | Wait until the file is closed         |
| `code --diff file1 file2` | Compare two files                     |
| `code --add folder`       | Add a folder to the current workspace |
